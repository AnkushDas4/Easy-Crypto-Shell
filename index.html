The Universal
Web3 Wrapper.

A single, configuration-driven API to read, write, and manage native coins and tokens across thousands of EVM networks and Solana. No heavy math, no complex ABIs.

Installation

Install the shell and its peer dependencies via NPM:

npm install easy-crypto-shell ethers @solana/web3.js @solana/spl-token bs58

Quick Start

Initialize the shell by passing the name of the chain you want to interact with. The engine handles the architecture routing automatically.

const { UniversalWallet } = require("easy-crypto-shell");

// Initialize chains instantly
const ethWallet = new UniversalWallet("ethereum");
const polyWallet = new UniversalWallet("polygon");
const solWallet = new UniversalWallet("solana");

1. Reading Native Balances

Fetch formatted balances without worrying about Wei, Gwei, or Lamports.

const vitalikAddr = "0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045";
const solTreasury = "vines1vzrYbzLMRdu58ou5XTby4qAqVRLmqo36NKPTg";

// Returns formatted string (e.g., "1.5")
const ethBal = await ethWallet.getBalance(vitalikAddr);
const solBal = await solWallet.getBalance(solTreasury);

2. Sending Transactions

Compile, sign, and broadcast transactions using the exact same method across all architectures.

// Sending Polygon (EVM)
const tx1 = await polyWallet.send(
    "YOUR_PRIVATE_KEY", 
    "0xDestinationAddress...", 
    "0.05"
);

// Sending Solana (SVM) - Base58 Private Keys are automatically decoded
const tx2 = await solWallet.send(
    "YOUR_PHANTOM_PRIVATE_KEY", 
    "DestinationBase58Address...", 
    0.05
);

3. Universal Token Engine

Read and send ERC20 and SPL tokens. For Solana, Associated Token Accounts (ATAs) are derived and created automatically behind the scenes.

// USDC Contract Addresses
const usdcPolygon = "0x3c499c542cEF5E3811e1192ce70d8cC03d5c3359";
const usdcSolana = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v";

// Read Token Balances
const polyUsdc = await polyWallet.getTokenBalance(usdcPolygon, vitalikAddr);
const solUsdc = await solWallet.getTokenBalance(usdcSolana, solTreasury);

// Send Tokens
await polyWallet.sendToken(privKey, usdcPolygon, toAddress, "10.0");
await solWallet.sendToken(privKey, usdcSolana, toAddress, 10.0);
